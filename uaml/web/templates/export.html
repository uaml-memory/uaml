📦 Export / Import

Data export, import, and backup management

📤 Export

CLI: uaml io export --layer knowledge --output data.jsonl

📥 Import

CLI: uaml io import data.jsonl

💾 Backup

Manage backups via CLI:

  • uaml backup run — Create backup
  • uaml backup list — List backups
  • uaml backup verify <path> — Verify backup integrity
  • uaml backup config — Configure backup (PQC encryption, retention)
